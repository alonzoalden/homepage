# alonzoalden.com

Static portfolio and resume site for Alonzo Alden, positioned as a Senior Front-End Engineer specializing in Angular, TypeScript, and complex enterprise applications.

## Structure

- `index.html` - Complete static website, including the HTML resume.
- `assets/hero-workspace.jpg` - Project-local hero image asset.
- `assets/Alonzo_Alden_Senior_Frontend_Engineer_Angular_2026.pdf` - Primary downloadable resume PDF.
- `assets/Alonzo_Alden_Senior_Frontend_Engineer_Angular_2026.docx` - Editable Word version of the primary resume.
- `CNAME` - Custom domain for GitHub Pages.

## Deploy

This site does not require a build step. Serve the repository root with any
static host, including GitHub Pages, Netlify, Vercel, Cloudflare Pages, or the
current hosting provider for `alonzoalden.com`.
